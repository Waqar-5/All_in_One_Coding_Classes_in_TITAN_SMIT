# 🎓 Student Management System

A simple and beginner-friendly Student Management System built using **HTML, CSS, and JavaScript**.  
This project allows users to add students with their name and age, and display them in a dynamic list.

---

## 📌 Features

- ✅ Add student name and age
- ✅ Input validation (prevents empty fields)
- ✅ Dynamic student list display
- ✅ Clean and simple UI
- ✅ Beginner-friendly code structure

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- DOM Manipulation
- Arrays
- Functions
- Loops
- Event Handling

## 🚀 How It Works

1. User enters **Student Name**
2. User enters **Student Age**
3. Clicks on **Add Student**
4. JavaScript:
   - Validates input
   - Adds student to array
   - Updates the student list dynamically
   - Clears input fields
